A3Cube Architecture

A3CUBE’s new breakthrough architecture is designed to provide enormous operational optimization and economic advantages in Big Data and HPC environments. Our unique vision adjusts the focus from High Performance Computing (HPC) to High Performance Data (HPD) with the aim of addressing current and future data center challenges. This revolutionary architecture is based on our "In Memory Network" with direct memory to memory communication, that uniquely extends the very limited capability of PCIe architectures and interconnection networks in general, to extol today’s and tomorrow’s flash technology potential. Technology has witnessed a radical architectural evolution in the last 20 years, particularly in high performance computing (HPC). The market has shifted from a few very powerful computational elements using the traditional supercomputer architectural model, to thousands of commodity computing elements, creating more affordable and scalable computing systems and making HPC available to a wider range of applications. Storage systems and data analytic machines must make the same transition from a few high performance storage engines to thousands of networked storage devices operating in parallel to deliver on the promise of real scale out storage. A3CUBE has now adapted this supercomputing architecture to enable exascale storage built around a ‘brain-inspired’ architecture. It combines supercomputing’s massively parallel operations with an innovative data plane and I/O fabric that eliminates network overhead while dramatically slashing the latency of traditional network fabric designs with new features The A3CUBE design encompasses breakthrough technology, creating a new standard for storage nodes that can be comprised of disks, SSDs, CPUs and I/O interfaces, becoming equivalent to a processing element in massively parallel computer environments. It leverages a uniquely powerful communication mechanism between these nodes called the RONNIEE™ "In Memory Network." This is a revolutionary paradigm shift in network communication, that introduces a full application with a transparent memory-to-memory direct connection. This In-Memory network discards the protocol stack bottleneck involved in remote memory access, resulting in the lowest possible communication latency, even for user-level software. The TCP/UPD stack is completely removed, not simply offloaded, and substituted with a fast memory-to-memory mapped TCP/UDP socket that produces extraordinary and disruptive performance enhancements at the application and system levels and allows applications to reach their full potential. Our technology introduces elite supercomputing performance and characteristics to every organization without any changes to the native applications.

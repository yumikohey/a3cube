FORTISSIMO Foundation

Fortissimo Foundation brings a new dimension to Storage and Converged computing. It is an innovative scalable clustered platform that offers linear scalability, a distributed front end and a global namespace without any limitations. FORTISSIMO is designed to provide a scale out in memory storage experience applied to data access and computing.

  • Leverage parallel design concepts
  • Intelligent relationship between available CPUs and I/O
  • Bottleneck-free no-metadata server and architecture
  • Scales capacity and I/O independently
  • Unique Scale Out RAM aggregation across all the nodes.
  • Automatic file replication, snapshots and undelete.
  • Support data deduplication, compression and encryption
  • Manages Data Plane and User Network separately
  • Accelerate remote devices access across all the nodes
  • Natively supports ultra-low latency
